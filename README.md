# Hedge Fund Time Series Forecasting

## Overview
This project builds predictive models for financial time-series data using Kaggle's Hedge Fund forecasting competition dataset.

## Objectives
- Explore and understand time-series structure
- Perform feature engineering
- Train predictive models
- Evaluate forecasting performance

## Notebooks
- exploratory-data-analysis.ipynb
- modeling.ipynb

## Tools
Python, Pandas, NumPy, Scikit-learn, Matplotlib

## Status
Model development in progress.
